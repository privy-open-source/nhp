# NHP

> Nuxt HTTP Proxy Module

## Compabilities

- Nuxt 3

## Instalation

```
yarn add --dev @privyid/nhp
```

Then, add into `nuxt.config.ts` modules

```ts
export default defineNuxtConfig({
  modules: ['@privyid/nhp/module'],
})
```
